      How this tool works

      Upload mode — pick the lecture you're mapping, drop the LO doc + any slides/readings, click Run. Claude reads the content, cross-checks it against the Endocrine block's 6 resource hierarchies (Bootcamp, B&B, Pathoma, First Aid, Physeo, Sketchy) and the UWorld QID catalog, and returns proposed primary / additional-primary / cross-reference assignments per resource plus matched UWorld QIDs. You review on-screen and download the updated xlsx files.

      Audit mode — lets you re-check a lecture against the baseline state. Uploads the current LO doc; if its SHA-256 matches the baseline you're done; if content changed, Claude re-maps and shows you the diff (added / removed subchapter assignments, new / removed QIDs). You can save an audit state file to preserve the baseline across sessions.

      Your API key is stored only in this browser's localStorage. Every request goes directly from your browser to Anthropic using the anthropic-dangerous-direct-browser-access header. Cost per lecture: roughly < $0.20 with Claude Sonnet.

      Downloads reproduce the same format as your original xlsx files: master mapping sheet with rich text (primaries in black, xrefs italic gray, consecutive xref subchapters collapsed into ranges) + 6 per-resource sheets, and the UWorld QID sheet with moccasin highlighting on high-risk QIDs.
